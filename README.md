# Data Analysis Agent

## Overview
This project is an Intelligent Data Analysis Agent that automates Exploratory Data Analysis (EDA) and basic machine learning.

## Features
- Upload CSV dataset
- Data preview
- Missing value analysis
- Summary statistics
- Correlation heatmap
- Outlier detection
- Automatic model training (Classification/Regression)
- Download processed data

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the application:
streamlit run app.py

## Dataset
You can use any CSV dataset (e.g., Titanic dataset from Kaggle).
